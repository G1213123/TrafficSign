Message Colour Selection

TPDM Vol.3 – Section 3.5.3, 3.5.4.6–3.5.4.7; Colour formats per 3.2.7.1

Choose text colour based on the panel/background

Select the message colour to maintain contrast with the sign panel (border/background). Use this table:

Background Type Typical Background Colours Message Colour Transport Alphabet
Dark background Green Blue Brown Red White Transport Medium
Light background White Yellow Black Transport Heavy

For bilingual signs, follow the same contrast rule for both languages.

Quick reference
  • Green/Blue panels (e.g., expressway, trunk): use White text.
  • White/Yellow panels (local/temporary): use Black text.
Notes

Refer to TPDM for full colour formats and panel rules. See also 3.5.7 for symbol colour behaviour on green panels.
