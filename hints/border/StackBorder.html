Stack Border

TPDM Vol.3 - Section 3.5.5 (Roadside Directional Signs), Section 3.5.6 (Gantry Signs), Section 3.5.4.8 (Sign Border Details), Section 3.5.4.16 (Border Rounding)

Usage

General border format for stack type, gantry type and map type sign.

Combine with Stack Divider when multiple horizontal blocks are needed.

Also can combine with Gantry Divider for vertical lane sectioning.
