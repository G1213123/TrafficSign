About Road Sign Factory

Road Sign Factory provides traffic engineers, designers, and enthusiasts with a standards-compliant tool for creating professional road signage. Built with modern web technologies and following Hong Kong TPDM guidelines.

Key Features

  • Standards Compliance: Built to comply with Hong Kong Transport Planning and Design Manual (TPDM) guidelines, ensuring professional-grade sign design.
  • Multiple Export Formats: Export your designs in SVG, DXF, and PDF formats for various applications and printing requirements.
  • Template Library: Access pre-designed templates for common road sign types including roundabouts, diversions, and directional signs.
  • Real-time Preview: See your changes instantly with our live preview system, making design iteration quick and efficient.
  • Customizable Chinese Fonts: Built-in support for customizable Chinese typography with professional fonts including TW-MOE-Std-Kai, ensuring authentic and compliant bilingual signage for Hong Kong standards.
  • Web-based Solution: No software installation required - works directly in your web browser with modern HTML5 technologies.
  • Open Source: Fully open-source project available on GitHub, allowing for community contributions and transparency.
  • Save & Load System: Automatic save to browser local storage ensures your work is never lost, with manual save and load options for project management and backup.

Technical Information

Road Sign Factory is built using modern web technologies including HTML5 Canvas, JavaScript ES6+, and responsive CSS. The application uses the Fabric.js library for advanced canvas manipulation and supports vector graphics export through SVG generation.

The application features an intelligent save system that automatically stores your work in the browser's local storage, preventing data loss. Additionally, manual save and load functionality allows you to manage multiple projects and create backups of your designs.

The export function supports multiple file formats including PNG (raster), SVG (vector), PDF (document), and DXF (CAD). PDF export is handled using the jsPDF library, which converts the canvas content to high-quality PDF documents suitable for printing and sharing. DXF export is powered by the dxf-writer library, enabling seamless integration with professional CAD software and engineering workflows. This allows your traffic sign designs to be easily imported into AutoCAD, SolidWorks, and other industry-standard design tools.
