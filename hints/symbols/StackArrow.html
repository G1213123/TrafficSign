TPDM Vol.3 - Section 3.5.5.22-3.5.5.31 (Stack Type Signs), Section 3.5.4.10 (Alignment), Section 3.5.4.15 (Symbol Spacing)

Stack Arrow Symbol Usage

Used in roadside stack-type directional signs for indicating direction of travel

Follows left-alignment principles appropriate for stack type signs (Section 3.5.4.10)

Spacing: 1.5 s/w between message and symbols, 2 s/w between symbols (Section 3.5.4.15)
