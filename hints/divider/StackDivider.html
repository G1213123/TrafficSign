Stack Border Divider

TPDM Vol.3 - Section 3.5.5.22-3.5.5.30 (Stack Type Signs), Section 3.5.4.10 (Alignment of Legend Blocks)

Stack Border Divider Usage

Used to separate content vertically within stack-type borders for roadside directional signs

Creates horizontal divisions between different sections of the sign, following left-alignment principles

Border spacing follows specifications in Table 3.5.4.1 with 1.5 s/w border width for directional signs

Select the divider when placing Stack border to assign the divider width

Design Reference

Spacing between legend blocks: 2.5 s/w from upper border, 1.5 s/w from lower border (Section 3.5.4.9)

Alignment: Left-alignment appropriate for stack type signs (Section 3.5.4.10)

Border dimensions subject to rounding as per Section 3.5.4.16
