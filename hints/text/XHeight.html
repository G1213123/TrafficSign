x-Height Input Guide

TPDM Vol.3 - Section 3.2 (Lettering Dimensions), Table 3.2.5.1

What is x-height?

x-height is the nominal height of lowercase letters in the Transport alphabet used on traffic signs. This value controls entire sizing of the sign design. It is used with stroke width, a basic unit in graphic design to measure length and width of elements, to obtain the actual dimension of the physical signs. Common relation between the value of xHeigh, stroke width and mm could be expressed as: mm = x-Height * stroke width / 4.

Minimum clear visibility

Table 3.2.5.1 also specifies minimum clear visibility distances for each case. Ensure siting achieves at least the stated minimums; strive for more where feasible (see 3.2.5.4–3.2.5.7).
